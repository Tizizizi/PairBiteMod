### 技术特点

✅ **零成本运行**

基于微信云开发，不需要服务器、域名、备案。免费额度（20 万次/月调用、3GB 容量、10GB CDN 流量）完全够两个人日常使用。

✅ **图片智能缓存**

客户端 90 分钟缓存 + 云函数分批处理，解决云存储免费环境权限限制，同一张图片不重复请求临时链接。

## 02 · 注册小程序账号

1. 打开 [微信小程序注册页](https://mp.weixin.qq.com/wxopen/waregister?action=step1)，选择「小程序」类型
2. 使用一个**未注册过微信公众平台的邮箱**，按引导完成注册，主体选「个人」
3. 注册完成后登录 [微信公众平台](https://mp.weixin.qq.com/)，在微信小程序后台首页，完成【小程序信息】和【小程序类目】的信息补充（小程序类目选 工具->备忘录 即可，并设置为主营类目）
4. 进入 `开发管理 → 开发设置`，记下你的 **AppID**（`wx` 开头的字符串）

> ⚠️ **注意**：一个邮箱只能注册一种平台账号，一个身份证最多 5 个个人小程序。

## 03 · 安装微信开发者工具

1. 下载 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html) 「稳定版」（Windows/Mac）
2. 安装后打开，用**小程序管理员的微信**扫码登录

## 04 · 导入项目

1. 将 `PairBite-mod.zip` 解压到电脑任意位置
2. 打开开发者工具 → 点「**+**」或「导入」
3. **目录**：选择解压后的 `PairBite-mod` 文件夹（里面应有 `miniprogram` 和 `cloudfunctions`）
4. **AppID**：填入你的 AppID
5. **后端服务**：选择「微信云开发」
<img src="images/9.png"  />

## 05 · 开通云开发环境

1. 点击开发者工具顶部的「**云开发**」按钮（云朵图标）

3. 付费方式选「按量付费」（有免费额度）

## 06 · 修改配置文件

只需要改两个文件：

### 6.1 修改 `project.config.json`

在项目根目录，用 `Ctrl+F` 搜索 `appid`，把值改成你自己的 AppID。
<img src="images/12.png"  />

### 6.2 修改 `miniprogram/envList.js`

把 `envId` 的值改成你自己的云环境 ID：

```js
const envList = [{"envId":"你的云环境ID","alias":"cloud"}]
```
<img src="images/13.png"  />

> ❤️ **就这两处！** 改完按 `Ctrl+S` 保存，其他代码都不用动。

## 07 · 部署 10 个云函数

在开发者工具左侧找到 `cloudfunctions` 文件夹，展开后看到 10 个子文件夹：

| 云函数 | 功能 |
|--------|------|
| `createUser` | 用户注册/登录 |
| `bindPartner` | 伴侣绑定 |
| `unbindPartner` | 解除绑定 |
| `getCoupleData` | 查询共享数据 |
| `updateCoupleData` | 更新共享数据 |
| `manageCategory` | 分类管理 |
| `updateKitchenName` | 厨房命名 |
| `sendNotify` | 消息通知 |
| `getOpenId` | 获取用户标识 |
| `getFileURL` | 图片链接转换 |

### 对每个云函数文件夹操作：

1. 先右键`cloudfunctions` 文件夹，选择环境
2. 右键文件夹 → 「**创建并部署：云端安装依赖（不上传 node_modules）**」
3. 等待控制台显示"上传云函数 xxx 成功"

<img src="images/15.png"  />

## 08 · 创建数据库集合

打开云开发控制台 → 「数据库」→ 点「+」添加集合，依次创建 4 个（名称区分大小写）：

```
User
DishList
OrderList
Category
```
<img src="images/16.png"  />

> 💡 **不需要手动建表结构**：只要创建空集合就行，字段由代码自动管理。首次绑定伴侣后，分类数据也会自动初始化。

## 09 · 配置订阅消息模板（可选）

如果需要点菜后推送微信通知给对方：

1. 登录 [微信公众平台](https://mp.weixin.qq.com/) → `功能 → 订阅消息 → 公共模板库`
2. 搜索「日程提醒」，选择包含以下字段的模板：**时间、任务名称、提醒内容、备注**，添加并提交
<img src="images/17.png"  />
<img src="images/18.png"  />
3. 回到「我的模板」，复制 **模板 ID**
4. 打开项目中的 `miniprogram/app.js`，搜索 `notifyTmplIds`，把值替换成你的模板 ID

> 💡 **不配置也不影响核心功能**，只是点菜后不会推送通知给对方。
>
> ⚠️ **收不到通知？** 微信订阅消息需要**双方都授权**才能互相收到。使用时遇到授权弹窗，请勾选「总是保持以上选择」并点击允许。

## 10 · 上传为体验版

1. 点顶部「**工具、上传**」按钮，填版本号（如 1.0.0）和描述
<img src="images/19.png"  />
2. 登录 [微信公众平台](https://mp.weixin.qq.com/) → `管理 → 版本管理`，在「开发版本」中找到上传的版本
3. 点「▼」下拉 → 「**选为体验版本**」，保存体验版二维码
<img src="images/20.png"  />

## 11 · 邀请对象使用

1. 微信公众平台 → `管理 → 成员管理 → 体验成员` → 添加对象的微信号
2. 对方确认邀请后，扫描体验版二维码打开小程序
3. **你**设置头像昵称 → 进入「绑定」页 → 获得邀请码
4. **对象**设置头像昵称 → 输入你的邀请码 → 绑定成功 🎉

## 12 · 个性化修改

### 修改小程序名称和图标

在 [微信公众平台](https://mp.weixin.qq.com/) → `设置 → 基本设置` 里修改（不是在代码里改）。

### 修改品牌名称

只需要改两个地方：

- `app.json` 的 `navigationBarTitleText`
- `app.js` 的 `appName`

### 修改主题色

`app.json` 的 `navigationBarBackgroundColor` 改成你喜欢的颜色。

### 更换装饰图片

替换 `miniprogram/images/` 目录下的同名文件即可，支持 jpg/png/gif。

## 13 · 隐私保障

✅ **三重保障**

- **不提交审核上架** — 只用体验版，搜索不到
- **体验成员白名单** — 只有你添加的人才能使用
- **邀请码绑定** — 数据按 coupleId 隔离，技术层面确保只有你俩看到

## 14 · 常见问题

**Q: 白屏或报"云环境不存在"**

检查 `miniprogram/envList.js` 的 envId 是否和云开发控制台顶部的环境 ID 一致。

---


**Q: 对象扫码提示无权限**

确认已在微信公众平台 → 成员管理 → 体验成员中添加了对方微信号且对方已确认。

---

**Q: 代码包超过 2MB**

用 [TinyPNG](https://tinypng.com) 压缩 `miniprogram/images/` 下的图片。

---

**Q: 会产生费用吗？**

免费额度（20 万次/月调用、3GB 容量、10GB CDN）完全够两个人使用。项目自带图片缓存优化，正常使用不会超限。

---

**Q: 修改代码后怎么更新？**

改了前端代码 → 重新上传体验版。改了云函数 → 右键重新部署。手机端重新打开即生效。
